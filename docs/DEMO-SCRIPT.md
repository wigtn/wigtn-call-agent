# Demo Script (2분)

> AI Call Agent 시연 스크립트

## Pre-Demo Checklist

- [ ] `npm run dev` 실행 중
- [ ] `.env.local` 설정 완료
- [ ] Mock 모드 확인: `ELEVENLABS_MOCK=true` (안전한 기본값)
- [ ] 브라우저 localhost:3000 열어두기
- [ ] 화면 녹화 시작 (백업용)

---

## Demo Flow (2분)

### Scene 1: 문제 소개 (20초)

> "전화 통화가 불편한 분들이 많습니다. 사회 불안, 언어 장벽, 청각 장애 등으로
> 간단한 미용실 예약이나 식당 문의도 어렵습니다.
> AI Call Agent는 AI가 대신 전화해주는 서비스입니다."

### Scene 2: 요청 입력 (20초)

**화면**: 메인 페이지 (localhost:3000)

1. 요청 입력란에 타이핑:
   > "내일 오후 3시에 OO미용실 커트 예약해줘"
2. 전화번호 입력:
   > "010-1234-5678"
3. [전화 부탁하기] 버튼 클릭

> "사용자는 이렇게 간단히 요청만 입력하면 됩니다."

### Scene 3: AI 파싱 & 확인 (20초)

**화면**: 확인 페이지 (/confirm/[id])

> "AI가 자연어를 분석해서 예약 정보를 추출합니다."

- 장소: OO미용실
- 날짜: 내일 (2/7)
- 시간: 오후 3시
- 서비스: 커트

> "확인 후 [전화 걸기]를 누르면 AI가 실제로 전화를 겁니다."

[전화 걸기] 버튼 클릭

### Scene 4: 통화 중 (20초)

**화면**: 통화 중 페이지 (/calling/[id])

> "지금 AI가 미용실에 전화를 걸고 있습니다.
> 경과 시간과 진행 상황을 실시간으로 보여줍니다."

(5초 대기 - mock 모드에서 자동 완료)

### Scene 5: 결과 확인 (20초)

**화면**: 결과 페이지 (/result/[id])

> "예약이 완료되었습니다!
> AI가 통화 결과를 요약해서 보여줍니다."

- 성공 체크마크
- 예약 정보 카드
- AI 요약 텍스트

### Scene 6: 마무리 (20초)

> "이렇게 전화 한 통 없이 예약이 완료됩니다.
> 기술 스택은 Next.js 16, ElevenLabs Conversational AI, OpenAI GPT-4입니다.
> 감사합니다."

## Fallback Plans

### Plan A: Full Demo (ElevenLabs 실제 통화)
- `ELEVENLABS_MOCK=false` 설정
- 팀원 실제 번호로 테스트
- AI가 실제 전화하는 것 시연

### Plan B: Mock Demo (기본값)
- `ELEVENLABS_MOCK=true` (default)
- 전체 플로우 동작하되 실제 전화는 안 감
- 5초 후 자동 완료

### Plan C: Screen Recording
- 사전에 녹화한 데모 영상 재생
- "라이브 환경에서는 실제 전화가 발신됩니다" 설명

### Plan D: Screenshots
- 각 화면 스크린샷 준비
- 슬라이드로 플로우 설명

2. **기술**: "ElevenLabs Conversational AI로 자연스러운 한국어 대화 가능"
3. **확장성**: "미용실뿐 아니라 식당, 병원, 카페 등 다양한 분야로 확장 가능"
4. **사회적 가치**: "전화 불안, 청각 장애, 외국인 등 사회적 약자 지원"
